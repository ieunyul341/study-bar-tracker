# 오늘한칸

공부한 시간을 과목별 막대에 채워가는 간단한 공부 기록 웹앱입니다.

## 주요 기능

- 월간 / 주간 보기 전환
- 국어, 수학, 영어, 과학, 사회 과목별 목표 막대
- 날짜별 공부 시간과 메모 기록
- 과목별 `+30분`, `+1시간` 빠른 기록
- 선택한 기간의 총 공부 시간, 공부한 날, 달성률 표시
- 브라우저 `localStorage` 자동 저장
- 모바일 반응형 화면

## 실행 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 저장소 이름을 `study-bar-tracker`로 정합니다.
3. `index.html`과 `README.md`를 업로드합니다.
4. 저장소의 **Settings → Pages**로 이동합니다.
5. **Deploy from a branch**를 선택합니다.
6. Branch를 `main`, 폴더를 `/root`로 지정하고 저장합니다.
7. 잠시 기다리면 공개 주소가 생성됩니다.

## 데이터 저장 주의

공부 기록은 현재 사용하는 브라우저에만 저장됩니다. 브라우저 데이터를 지우거나 다른 기기를 사용하면 기록이 보이지 않습니다.
