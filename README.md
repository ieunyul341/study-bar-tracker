# 오늘한칸 v2

공부 시간, 습관, 투두리스트를 한 화면에서 관리하는 웹앱입니다.

## 추가된 기능

- 국어, 수학, 영어, 과학, 사회 과목별 서로 다른 색상
- 물 마시기, 운동, 양치, 책 읽기 습관 체크
- 주간 및 월간 습관 달성률 표시
- 지난 주와 지난 달 기록 다시 보기
- 날짜별 작은 투두리스트
- 투두 완료 체크 및 삭제
- 기존 v1 공부 기록 자동 불러오기
- 브라우저 자동 저장

## GitHub 업데이트 방법

기존 저장소의 `index.html` 파일을 이 버전의 `index.html`로 교체하면 됩니다.

1. 저장소에서 기존 `index.html`을 클릭합니다.
2. 오른쪽 위 연필 아이콘을 클릭합니다.
3. 기존 내용을 모두 지우고 새 코드를 붙여 넣거나,
4. 더 쉬운 방법으로 기존 `index.html`을 삭제한 뒤 새 파일을 업로드합니다.

## 주의

기록은 브라우저 localStorage에 저장됩니다. 같은 기기와 같은 브라우저에서 이용해야 기존 기록을 계속 확인할 수 있습니다.
